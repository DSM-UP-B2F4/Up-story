## Up-story 기능명세

### 푸터

- <메뉴가 궁금할 땐>
  - `스토리` : 클릭 시 스토리 페이지로 이동.
- <정책이 궁금할 땐>
  - 이용약관, 오픈 API 이용약관 클릭 시 각 페이지 이동.
- 검색어 입력 창 : 검색어 입력 후 검색 버튼 또는 엔터를 누르면 다음 페이지로 이동해서 검색됨.



### 로그인 페이지

아이디 입력 하는 창이 뜸. 

- `로그인 버튼` : 아이디와 비밀번호가 맞으면 로그인이 됨. 일치하지 않을 시 일치하지 않는다는 메세지가 뜸.
- `회원가입` : 회원가입 페이지로 이동.
- `아이디/비밀번호 찾기` : 아이디/비밀번호 찾기 페이지로 이동.



### 메인페이지

#### 로그아웃 시

- `티스토리 로고` : 클릭 시 메인 페이지로 이동. 

- `시작하기` : 클릭 시 로그인 페이지로 이동.
- `피드` : 클릭 시 로그인 페이지로 이동.
- `스토리` : 클릭 시 스토리 페이지로 이동.

#### 로그인 시

* `티스토리 로고` : 클릭 시 메인 페이지로 이동. 
* `피드` : 클릭 시 피드 페이지로 이동.
* `스토리` : 클릭 시 스토리 페이지로 이동.
* `알림` : 클릭 시 새소식 확인 창 생성.
* `프로필` : 클릭 시 닉네임, 이메일, 계정관리, 로그아웃이 뜬 창 생성.
  * 계정관리, 설정버튼 : 클릭 시 계정관리 페이지로 이동.
  * 운영중인 블로그 이름 : 클릭 시 운영 중인 블로그 페이지로 이동.
  * 연필 버튼 : 클릭 시 글쓰기로 이동.
  * 로그아웃 : 클릭 시 로그아웃.
* 사진 : 마우스 커서가 있을 때 검정색 배경에 설명이 뜸. 클릭 시 해당 주소로 이동.
* 화살표버튼 :  마이스 커서가 근처에 가면 생김, 마우스 커서가 있을 때 주황색. 클릭 시 방향에 맞게 이동.
* 기업 로고 : 클릭 시 해당 사이트로 이동.



### 계정관리 페이지

- <내 계정>

  - 기본 정보 : 닉네임과 소개 칸 클릭 후 변경 가능. 

    `변경사항 저장` : 변경사항이 저장됨.

    `티스토리에서 탈퇴하기` : 티스토리에서 탈퇴됨.

- <보안>

  - 아이디 변경하기 : 현재 비밀번호 입력 후 `확인하기` 버튼 클릭하면 아이디 변경 창으로 이동. `변경하기` 클릭 시 변경됨.
  - 비밀번호 재설정 : 현재 비밀번호 입력 후 `확인하기` 버튼 클릭하면 비밀번호 변경 창으로 이동. `변경하기` 클릭 시 변경됨.

- <내 블로그>

  * 블로그 이름, 닉네임, 블로그 설명 : 사용자 입력받아 변경 가능.

  * 사진 추가 : 라이브러리에서 사진 추가 가능.

    `변경사항 저장` : 변경사항이 저장됨.

  * 데이터 관리 : 클릭 시 데이터 관리 페이지 이동.

    <데이터 관리>

    `삭제하기` : 클릭 시 현재 비밀번호 입력.

    - 5회 실패 시 1시간 후 재입력 가능.
    - `확인` : 클릭 시 블로그 데이터 모두 삭제.

    <블로그 폐쇄>

    `폐쇄하기` : 클릭 시 블로그 폐쇄 확인 창 뜸.  취소 버튼 누를 시 취소됨. 확인 버튼 누를 시 블로그 폐쇄.

    

### 피드 

* (구독중이 0명일 시) `스토리 보러가기` : 스토리 페이지로 이동.
* `구독중` : following 페이지로 이동.
* `구독자` : follower 페이지로 이동.
* `구독 추천 박스` : 블로그 홈페이지 이동. `구독하기` 클릭 시 구독.



### 스토리

- 라이프, 여행.맛집, 문화.연예, IT, 스포츠, 시사가 있음.
- 각각 주제를 누르면 그 카테고리를 선택한 글들이 시간 순서대로 나옴.
- 블로그 박스를 누르면 그 블로그의 게시글로 넘어가짐.
- `구독하기` : 페이지가 다시 로드되고, 구독됨.
  - 로그인이 안 되어있을 시 로그인 페이지로 이동.
- 스크롤이 끝없이 계속 로드된다.



### 블로그 페이지 

#### 상단 

* 가운데 블로그 이름 : 클릭 시 블로그 페이지 다시 로드됨.

* 검색 버튼 : 마우스 커서가 있을 때, 검색 버튼 주변으로 박스가 생기면서 회색 배경 생성. 클릭 시 내용을 입력할 수 있는 박스 생성. 내용 입력 후 검색 또는 엔터 누르면 검색됨.

  - 검색한 내용이 내 블로그 게시글에 없는 경우 : 게시물이 아무것도 없는 페이지로 이동

  * 검색 내용이 내 블로그 게시글에 있는 경우 : 검색된 단어가 들어있는 게시물이 뜨는 페이지로 이동

#### 글 부분

* 사진이나 제목 : 클릭 시 게시글 페이지로 이동.

#### 하단

* 관리자 버튼 : 관리자 버튼을 누를 시 계정 관리 페이지의 <내 블로그> 페이지로 이동.

  - `삭제` : 클릭 시 확인 버튼 생성. `확인` 클릭 시 글 삭제.

  * `수정` : 클릭 시 (수정) 게시글 쓰기 페이지로 이동. 
  * `비공개 상태` : 클릭 시 글 공개 전환.
  * `공개 상태` : 클릭 시 글 비공개 전환.

* 댓글 박스 : 사용자로부터 댓글 입력받음. 로그아웃하고 댓글 박스 클릭 시 로그인 페이지로 이동.

* 댓글 쓰기 : 대댓글을 쓸  수 있음.

* 수정/삭제 

  * 본인이 쓴 글 : 수정 및 삭제 가능.
  * 남이 쓴 글 : 삭제만 가능.

  - 라이프, 여행.맛집, 문화.연예, IT, 스포츠, 시사가 있음.
